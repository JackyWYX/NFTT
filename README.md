# install
`yarn` or `npm install`

# generate typescripts
run `npx hardhat compile`, then it will generate typescripts in ./typechain

# compile
1. `npm install -g truffle`
2. `truffle compile`

# deploy
`truffle --network=xxx exec deploy_ipp.js`

# deploy local
1. `truffle develop`
2. `truffle --network=develop exec deploy_ipp.js`
